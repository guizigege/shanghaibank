支行网点

1. 自动获取网点已发布的业务（优品/活动），各业务按发布时间倒序展示，至少关联一个业务；

2. 若红包关联各业务的红包发放上限之和小于红包发放个数，则多出来的红包将连同客户未领完的红包金额，在该批业务全部下线后1个工作日内原来返回；

3. 单个业务的红包上限不能大于红包发放个数，当关联业务红包上限之和大于红包发放个数时，则各业务的具体发放个数由：红包发放个数、业务红包上限、客户领取情况共同决定，即客户进入某业务详情页领取红包时，只要尚有未领取的红包，且该业务下客户已领取的红包数量小于设置的红包上限，客户就可以看到红包标识，并顺利领取红包；

4. 若未设置业务的红包上限，则表示此红包不发放至该业务；

5. 若客户尚未关注该网点或客户经理，则领取红包后，自动关注。
